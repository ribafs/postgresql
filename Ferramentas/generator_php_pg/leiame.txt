Gerador de Aplicativos em PHP com PostgreSQL

Com a finalidade de auxiliar no aumento da produtividade na construção de sistemas web em PHP com PostgreSQL.

Testado em Linux e Windows

Em Linux requer que o usuário do apache seja o dono do diretório. No Apache2 algo como:
chown -R nobody:root gerador_phppg

Instalação:

Apenas descompacte num diretório do Servidor Web.
Abra a index.php no browser http://127.0.0.1/gerador_phppg/. Entre com os dados e os arquivos serão gerados.

Para testar o aplicativo gerado, abra no browser o diretório criado:

http://127.0.0.1/gerador_phppg/diretorio

Tested in Linux and Windows 

In Linux it requires that user of the apache is the owner of the directory.

chown - R nobody:root gerador_phppg

Just unpacks in a directory of the Web Server.  Http://127.0.0.1/gerador_phppg/.

http://127.0.0.1/gerador_phppg/directory
