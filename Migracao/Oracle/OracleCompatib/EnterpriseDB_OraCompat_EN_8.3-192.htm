7.1 DBMS_ALERT

The DBMS_ALERT package provides the capability to register for, send, and receive alerts.

The procedures and functions available in the DBMS_ALERT package are listed in the following table.

Table 7-1 DBMS_ALERT Functions/Procedures

Function/Procedure

Return Type

Description

REGISTER(name)

n/a

Register to be able to receive alerts named, name.

REMOVE(name)

n/a

Remove registration for the alert named, name.

REMOVEALL

n/a

Remove registration for all alerts.

SIGNAL(name, message)

n/a

Signals the alert named, name, with message.

WAITANY(name OUT, message OUT, status OUT, timeout)

n/a

Wait for any registered alert to occur.

WAITONE(name, message OUT, status OUT, timeout)

n/a

Wait for the specified alert, name, to occur.
