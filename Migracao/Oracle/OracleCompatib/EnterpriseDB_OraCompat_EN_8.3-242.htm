9.5 ALL_TABLES

The ALL_TABLES view provides information on all user-defined tables.

Name

Type

Description

owner

VARCHAR2

User name of the table’s owner.

schemaname

VARCHAR2

Name of the schema in which the table belongs.

table_name

VARCHAR2

Name of the table.

table_space

VARCHAR2

Name of the tablespace in which the table resides if other than the default tablespace.

status

VARCHAR2

Whether or not the state of the table is valid. Currently, always set to VALID.
