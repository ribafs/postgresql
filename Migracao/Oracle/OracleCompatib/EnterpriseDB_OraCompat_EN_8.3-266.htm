10.3.1 Enabling DRI

To switch on the collection of timing data, a new initialization parameter, timed_statistics, has been added. This is a dynamic parameter that can be set at the session as well as the system level in the postgresql.conf file. The valid values are TRUE or FALSE, with FALSE being the default. Data is collected at the beginning and end of each process and the deltas are calculated. At this time, individual statement data is not collected.
