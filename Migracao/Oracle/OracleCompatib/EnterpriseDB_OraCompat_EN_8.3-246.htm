9.9 DBA_ROLE_PRIVS

The DBA_ROLE_PRIVS view provides information on all roles that have been granted to users. A row is created for each role to which a user has been granted.

Name

Type

Description

grantee

VARCHAR2

User name to whom the role was granted.

granted_role

VARCHAR2

Name of the role granted to the grantee.

admin_option

VARCHAR2

YES if the role was granted with the admin option, NO otherwise.

default_role

VARCHAR2

YES if the role is automatically enabled when the grantee creates a session, NO otherwise. Currently, always set to YES.
