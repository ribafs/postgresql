9.3 ALL_SYNONYMS

The ALL_SYNONYMS view provides information on all synonyms.

Name

Type

Description

owner

VARCHAR2

User name of the synonym’s owner.

synonym_name

VARCHAR2

Name of the synonym.

object_owner

VARCHAR2

User name of the object’s owner on which the synonym is defined.

object_name

VARCHAR2

Name of the object on which the synonym is defined.

synacl

VARCHAR2

Not currently used – always set to null.

status

VARCHAR2

Whether or not the state of the synonym is valid. Currently, always set to VALID.
