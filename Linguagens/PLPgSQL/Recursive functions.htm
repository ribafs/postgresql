Factorial sample (PostgreSQL 7.3.x) 

In 7.3, recursion works fine. This is just a sample for 
beginner programmers. Note that any recursive function must 
check loop termination condition, or it will run forever 
(actually, until the process stack is full, but this is 
another story...).



create or replace function factorial(int) returns int as '
declare
        a int;
        ret int;
begin
        a := $1 - 1;

        if a = 0 then
                ret := $1;
        else
                ret := $1 * factorial(a);
        end if;

        return ret;
end;
' language 'plpgsql' immutable strict;
